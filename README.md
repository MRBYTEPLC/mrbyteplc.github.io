# MrBytePLC — Industrial Automation Intelligence

Personal website for [@MrBytePLC](https://www.youtube.com/@MrBytePLC) — deep technical content on
Studio 5000, OEE, MES, EtherNet/IP, and open-source PLC methodologies.

## Structure

```
mrbyteplc/
├── index.html              # Main homepage
├── pages/
│   └── insights.html       # STEM insights / blog articles
├── assets/
│   ├── css/
│   │   ├── main.css        # Design tokens, nav, footer, modal, animations
│   │   ├── hero.css        # Hero section & terminal widget
│   │   └── sections.css    # Pillars, mission, projects, roadmap, open source, subscribe
│   └── js/
│       └── main.js         # Video data, filtering, modal, scroll animations
└── README.md
```

## Features

- **Projects section** — YouTube-linked engineering projects with benefit lists, category filters, and modal player
- **Insights** — STEM-focused technical articles referencing open educational resources
- **Roadmap** — 4-phase technical development plan
- **Open Source** — GitHub repository cards
- **Mobile-responsive** with hamburger navigation
- **Add / remove projects** via built-in form (stored in localStorage)

## Deployment

Upload all files to any static host (GitHub Pages, Netlify, Vercel) maintaining the folder structure.

For GitHub Pages:
1. Push to a public repo
2. Go to Settings → Pages → Deploy from branch → `main` / root
